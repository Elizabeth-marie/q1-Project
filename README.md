# q1-Project
![image](https://user-images.githubusercontent.com/40966768/45891545-1544dc80-bd83-11e8-9a73-843966796ff3.png)

This Project allows you to find random quotes based on various categories for when you need inspiration!  
To run this in your browser, fork and clone the repo and open index.html in your browser.   
On the Quotes page, pick a category from the dropdown for a random quote to appear.  
To save to your favorites, click the heart button above the quote.
